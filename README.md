# Arithmetics

A collection of Python scripts demonstrating fundamental arithmetic operations, unit conversions, and mathematical calculations.

## Files

### `functions.py`
Contains utility functions and operations including:
- **Variable Swapping**: Swap values between two variables using a temporary variable
- **Random Number Generation**: Generate random numbers between 1 and 100
- **Unit Conversions**: 
  - Kilometers to Miles conversion (using factor 0.621371)
  - Celsius to Fahrenheit temperature conversion (formula: F = C × 9/5 + 32)

### `advance_calculations.py`
Contains more advanced arithmetic operations:
- **Addition**: Add two numbers and display the result
- **Division**: Divide two numbers with zero-division error handling
- **Triangle Area Calculation**: Calculate the area of a triangle using base and height (formula: Area = 0.5 × base × height)

## Usage

Each script is interactive and prompts the user for input. Run any script with Python:

```bash
python functions.py
python advance_calculations.py
```

## Features

- ✅ Interactive user input
- ✅ Error handling (e.g., division by zero)
